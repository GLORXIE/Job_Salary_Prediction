# Job Salary Prediction: Data Quality Assessment & Preprocessing

This repository contains a comprehensive data preprocessing pipeline designed to prepare raw job market data for machine learning. The project focuses on handling common real-world data issues, including missing values, outliers, and feature scaling.

## Overview
Before building a predictive model, data must be cleaned and standardized. This project demonstrates an end-to-end workflow to transform a "noisy" dataset into a high-quality input for regression algorithms.
